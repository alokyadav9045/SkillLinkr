🔗 SkillLinkr – Swap. Learn. Grow.
SkillLinkr is a community-driven skill exchange platform that connects people within a 5–10 km radius to learn and teach real-life skills — without money. Whether you’re a React developer wanting to learn guitar or a photographer seeking yoga tips, SkillLinkr helps you connect, exchange, and grow — right from your neighborhood.# 🌟 SkillLinkr


**Empowering Skill Exchange, One Connection at a Time!**

SkillLinkr is a modern skill-swapping platform that allows users to offer and learn new skills by connecting with others nearby. Whether you're a coder looking to learn guitar or a photographer willing to teach yoga, SkillLinkr brings people together for meaningful, skill-based collaboration.

---

## 🚀 Features

- 🔄 **Skill Exchange System** – Offer what you know, request what you want to learn.
- 🌍 **Nearby User Discovery** – Connect with users within 5-10 km radius.
- 📆 **Swap Requests** – Schedule requests with optional time/date slots.
- 🧵 **Real-time Chat** – Communicate with skill partners before you meet.
- ⭐ **Reviews & Ratings** – Build trust with skill-based reviews.
- 🧭 **Modern UI** – Mobile-first, sleek interface using TailwindCSS.
- 🛡️ **Secure Authentication** – JWT-based login/signup with profile management.

---

## 🖥️ Tech Stack

| Layer | Technology |
